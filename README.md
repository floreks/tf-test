# tf-test

This repository contains infrastructure-as-code examples using Terraform and Ansible.

## Terraform Configurations

- **Basic:** Generates a random string using the  resource.
- **Google Compute Instance:** Provisions a Google Cloud Compute instance with a custom service account.
- **Long:** Generates multiple random strings and outputs them.

## Ansible Playbooks

- **Basic:** A simple playbook that pings localhost and prints a message.
- **Variables:** Demonstrates usage of variables in Ansible playbooks.

## Usage

- Ensure you have Terraform and Ansible installed.
- Configure your cloud provider credentials as needed.
- Run Terraform configurations with  and .
- Run Ansible playbooks with  command.
