# tf-test

This repository contains Terraform and Ansible scripts for infrastructure management.

## Directory Structure

*   `ansible/`: Contains Ansible playbooks.
    *   `basic/`: A basic Ansible playbook.
    *   `variables/`: Variables for Ansible playbooks.
*   `terraform/`: Contains Terraform configurations.
    *   `basic/`: A basic Terraform configuration.
    *   `google-compute-instance/`: A Terraform configuration for a Google Compute Engine instance.
    *   `long/`: A long Terraform configuration.
    *   `once/`: A Terraform configuration that is only applied once.
    *   `variables/`: Variables for Terraform configurations.
