# Infrastructure Codebase

This repository contains basic examples of Infrastructure as Code (IaC) using Ansible and Terraform.

## Contents

### Ansible
- **Basic Playbook**: A simple playbook to ping hosts and print a message.
- **Variables**: Examples of using variables in Ansible.

### Terraform
- **Basic**: A simple Terraform configuration generating a random string.
- **Google Compute Instance**: Configuration for provisioning a VM on Google Cloud Platform.
- **Variables**: Demonstrates variable usage in Terraform resources.
