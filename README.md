# tf-test

This repository contains infrastructure automation configurations using Ansible and Terraform.

## Directory structure

### ansible/
Contains Ansible playbooks and variable files for configuration management.
- basic/: Basic Ansible playbooks.
- variables/: Ansible variable definitions.

### terraform/
Contains Terraform configurations for provisioning cloud infrastructure.
- basic/: Basic Terraform configurations.
- google-compute-instance/: Terraform configs for provisioning Google Compute instances.
- long/: More extensive Terraform configurations.
- variables/: Terraform variable files.

## Usage

- Use the Ansible playbooks to manage software configurations and deployments.
- Use the Terraform files to provision and manage cloud resources.
