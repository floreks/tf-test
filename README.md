# tf-test

This repository contains infrastructure as code configurations using Ansible and Terraform.

- The ansible/ directory holds Ansible playbooks and variable files.
- The terraform/ directory holds Terraform configurations for various compute instances and resources.

Usage:
- Use Ansible to automate setup tasks defined in ansible/basic/main.yaml.
- Use Terraform to manage infrastructure with the configurations under terraform/.

Make sure to configure variables and environment before running these automation scripts.

For issues or questions, please open an issue or contact the maintainer.
