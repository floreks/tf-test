# tf-test

## Repository Structure

This repository contains infrastructure as code configurations using Ansible and Terraform.

### Ansible
- `ansible/basic/main.yaml`: Basic Ansible playbook.
- `ansible/variables/main.yaml`: Ansible playbook for managing variables.

### Terraform
- `terraform/basic/main.tf`: Basic Terraform configuration.
- `terraform/google-compute-instance/main.tf`: Terraform configuration for Google Compute Instance.
- `terraform/long/main.tf`: Extended Terraform configuration.
- `terraform/variables/main.tf`: Terraform configuration related to variables.
- `terraform/variables/variables.tf`: Terraform variables definitions.

This structure supports managing infrastructure deployments and configurations using both Ansible and Terraform tools.
