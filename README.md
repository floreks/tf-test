# tf-test

This repository contains Terraform configurations and Ansible playbooks for infrastructure provisioning and automation.

## Terraform

- Basic random string generation and null resource example.
- Google Compute Engine instance provisioning with service account setup.
- Long-running Terraform example with multiple random strings and outputs.

## Ansible

- Basic playbooks for testing connectivity and printing debug messages.

## Usage

1. Terraform: Navigate to the desired Terraform directory and run `terraform init` and `terraform apply`.
2. Ansible: Run playbooks using `ansible-playbook` command targeting localhost or other hosts.

## Structure

- `terraform/`: Contains Terraform configurations.
- `ansible/`: Contains Ansible playbooks.

## Requirements

- Terraform installed for infrastructure provisioning.
- Ansible installed for configuration management.
