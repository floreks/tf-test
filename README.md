# TF Test Project

This repository contains Terraform and Ansible configurations for test infrastructure deployments.

## Directory Structure

- `ansible/`: Contains Ansible playbooks and variable configurations.
- `terraform/`: Contains Terraform infrastructure as code projects and variable definitions.

### Ansible
- `basic/`: Basic example playbooks.
- `variables/`: Variable definitions for Ansible playbooks.

### Terraform
- `basic/`: Basic Terraform configurations.
- `google-compute-instance/`: Resources for Google Compute instances.
- `long/`: Longer example Terraform configuration.
- `variables/`: Terraform variable definitions.

## Usage

- Use Ansible playbooks to provision and configure systems.
- Use Terraform configurations to create infrastructure resources.

## Notes

- Ensure correct variable values are set in respective files before applying configurations.
- Modify and extend playbooks and configurations as needed.
