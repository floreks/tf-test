# tf-test

This repository contains infrastructure-as-code configuration files using Ansible and Terraform.

## Repository Structure

### Ansible
- `ansible/basic/main.yaml`: Basic Ansible playbook.
- `ansible/variables/main.yaml`: Variables for Ansible playbooks.

### Terraform
- `terraform/basic/main.tf`: Basic Terraform configuration.
- `terraform/google-compute-instance/main.tf`: Terraform configuration for provisioning a Google Compute Instance.
- `terraform/long/main.tf`: A longer or more complex Terraform configuration.
- `terraform/variables/main.tf` and `terraform/variables/variables.tf`: Terraform variable definitions and configurations.

## Usage

- Use Ansible playbooks to manage configuration and deployment.
- Use Terraform files to provision and manage cloud infrastructure.
