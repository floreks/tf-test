# Infrastructure as Code Examples

This repository contains basic examples and tests for Infrastructure as Code (IaC) tools, specifically Ansible and Terraform.

## Structure

### Ansible
- **basic**: A simple playbook to ping localhost and print a message.
- **variables**: Demonstrates the use of variables in Ansible playbooks.

### Terraform
- **basic**: A simple configuration creating a random string and using a null_resource.
- **google-compute-instance**: Configuration to provision a Google Compute Engine instance with a specific service account and startup script.
- **variables**: Demonstrates the use of input variables to control resource properties.
