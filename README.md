# Infrastructure Code Examples

This repository contains a collection of basic examples and tests for **Ansible** and **Terraform**. It serves as a playground for testing infrastructure-as-code configurations.

## Contents

### Ansible
- **Basic Playbook**: A simple playbook to ping localhost and print a message.
- **Variables Playbook**: Demonstrates usage of variables in tasks.

### Terraform
- **Basic**: A configuration that generates a random string and echoes it using a local provisioner.
- **Google Compute Instance**: Provisions a Google Compute Engine instance and a Service Account with specific configurations.
- **Variables**: Shows how to define and use input variables.
