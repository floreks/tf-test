# tf-test

This project contains example configurations for Ansible and Terraform.

## Ansible

The Ansible directory contains playbook examples:
- `basic`: Simple playbook to ping hosts and print messages.
- `variables`: Demonstrates usage of variables in playbooks.

## Terraform

The Terraform directory includes several configurations:
- `basic`: Basic example to generate a random string and run a local-exec.
- `google-compute-instance`: Example setup to create a Google Compute Instance.
- `long`: Terraform setup with variable configurations for testing.
- `variables`: Variable definitions and example usage.

Ensure to set your variables and credentials accordingly before applying Terraform configurations.

---

These examples serve as templates for infrastructure testing and learning.
