# tf-test

This repository contains infrastructure-as-code configurations using both Ansible and Terraform.

## Repository Structure

### Ansible
- **ansible/basic/**: Contains basic Ansible playbooks.
- **ansible/variables/**: Contains variable definitions for Ansible playbooks.

### Terraform
- **terraform/basic/**: Basic Terraform configuration files.
- **terraform/google-compute-instance/**: Terraform configurations for Google Compute instances.
- **terraform/long/**: Extended or larger Terraform configurations.
- **terraform/variables/**: Variables and their definitions for Terraform configurations.

## Usage

- Use Ansible playbooks to automate system configuration and management.
- Use Terraform for provisioning and managing cloud infrastructure.

## Notes

- Make sure you have the required dependencies installed such as Ansible and Terraform CLI tools.
- Review each directory's README (if available) for more specific usage instructions.
