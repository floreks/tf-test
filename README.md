# Infrastructure Code Examples

This repository contains basic examples and test configurations for Infrastructure as Code (IaC) tools, specifically Ansible and Terraform.

## Contents

### Ansible
- **Basic Playbook**: A simple playbook located in `ansible/basic` that demonstrates running tasks on `localhost`.

### Terraform
- **Basic Configuration**: A simple Terraform setup in `terraform/basic` that generates a random string and uses a local-exec provisioner.
- **Google Compute Instance**: A configuration in `terraform/google-compute-instance` for provisioning a Google Compute Engine instance and a Service Account.
