# tf-test

tf-test is a small repository containing example Terraform and Ansible configurations for learning and testing.

Repository structure:
- ansible/: Ansible playbooks (basic, variables)
- terraform/: Terraform examples (basic, google-compute-instance, long, variables)

Quick start:
- Terraform: cd terraform/<example> && terraform init && terraform apply
- Ansible: ansible-playbook ansible/basic/main.yaml
